# EU Export Compliance Checker

A bilingual (English/Urdu) pre-shipment compliance checklist tool for Pakistani textile exporters to the European Union.

## Features

- **5 Compliance Sections** covering REACH, Customs Documentation, Labeling, Sustainability, and Import Formalities
- **31 Checklist Items** with critical/non-critical tagging
- **Bilingual Support** - English and Urdu (اردو)
- **Auto-Save** - Progress saved automatically in browser
- **Printable PDF Guide** - Click "Print PDF" to access the complete compliance guide
- **Responsive Design** - Works on desktop and mobile
